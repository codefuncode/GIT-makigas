Comandos usados en el tutorial 

===============================
Si escribimos en comando git en la consola y tenemos git instalado nos devolverá 
un listado de comandos que podemos utilizar en nuestro proyecto. 

Los siguientes comandos se usan generalmente la principio, estos para incorporar git a nuestro proyecto 


-------------------------------------------------------------
{git init}
Crea un repositorio de Git vacío o reinicia el que ya existe
-------------------------------------------------------------


------------------------------------------------------------
{git add}
Agrega contenido de carpetas al índice
------------------------------------------------------------

-----------------------------------------------------------
{git status}
Muestra el estado del árbol de trabajo
-----------------------------------------------------------

-----------------------------------------------------------
{git commit}
Graba los cambios en tu repositorio
-----------------------------------------------------------


**** Para realizar un commit usamos:
------------------------------------
{git commit -m "mensaje del commit"}
Esto evita abrir el editor por defecto en tu sistema
------------------------------------
