Debian / Ubuntu
Para la última versión estable para su lanzamiento de Debian / Ubuntu
-----------------------
# apt-get install git
-----------------------
Para Ubuntu, este PPA proporciona la última versión estable de Git ascendente

--------------------------------------------------------------------
# add-apt-repository ppa:git-core/ppa # apt update; apt install git
